# MacOS & Windows Guide

Before we get started it's best to check that your system meets the minimum requirements for the Unity Editor.  
  
  1. Check the minimum requirements for the Unity Editor [here](https://docs.unity3d.com/Manual/system-requirements.html#editor)  
  2. Check your system info and make sure it meets requirements
     - [View System info for Windows](https://support.microsoft.com/en-us/windows/view-your-system-info-a965a8f2-0773-1d65-472a-1e747c9ebe00)
     - [View System info for MacOS](https://support.apple.com/en-gb/guide/mac-help/syspr35536/14.0/mac/14.0)

Installation is pretty straight forward on both MacOS and Windows, follow the steps below:

<Steps>
  ### Step 1: Create a Unity ID

  You can register for Unity [here](https://id.unity.com/en/conversations/adc07ba5-7f26-4ad9-a8e0-069d3cd78fec007f),
  if you already have an account, you skip this.

  ![Image of Create Unity Account](https://support.unity.com/hc/article_attachments/12302710558484)

  ### Step 2: Install the Unity Editor & Modules

  1. Download the Unity Hub from [https://unity.com/download](https://unity.com/download)
  2. Sign in to your Unity ID by selecting the drop down at the top left corner of the screen.
    [Image of Sign-in]
  3. Select **Unity 2023.3.5f1 LTS**
    [Image of version selector]
    <Callout emoji="ðŸ’¡">
      Add the Microsoft Visual Studio Community by selecting the checkbox.
    </Callout>
  4. Click on the Install button at the right bottom corner of your screen.
    [Image of Install screen]
</Steps>
